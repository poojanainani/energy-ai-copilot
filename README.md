# ⚡ AI-Powered Energy Analytics Copilot

An AI system that converts natural language questions into SQL, runs them on PostgreSQL, and generates insights using an LLM (Phi-3 via Ollama).

## Features
- Natural language to SQL
- PostgreSQL integration
- AI insights
- FastAPI backend
- Streamlit frontend

## Tech Stack
- Python
- FastAPI
- Streamlit
- PostgreSQL
- Ollama (Phi-3)

## Example Questions
- Total energy usage by year
- Top 5 postal codes
- Highest district consumption
